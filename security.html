Security FAQ

Common security
questions

How does Payvexis encrypt my data?

All data is protected with 256-bit AES encryption, both in transit (TLS 1.3) and at rest. Card numbers are tokenized and never stored in plain text. Encryption keys are managed through hardware security modules (HSMs) that meet FIPS 140-2 Level 3 standards.

What happens if suspicious activity is detected?

Our AI systems automatically block the suspicious transaction, freeze the affected card if necessary, and send you an instant notification. Our security team investigates every flagged event and will contact you directly if additional verification is needed.

Is my biometric data stored on Payvexis servers?

No. Biometric authentication (Face ID, fingerprint) is processed and stored exclusively on your device using the manufacturer's secure enclave. Payvexis never receives, transmits, or stores your biometric data.

What security certifications does Payvexis hold?

Payvexis is PCI DSS Level 1 certified, SOC 2 Type II audited, GDPR compliant, and ISO 27001 certified. We undergo regular third-party penetration tests and publish our security audit results annually.
